# Course Management Web App (Next.js + Firebase)

##  Project Description

This is a Course Management web application built using Next.js and Firebase Authentication. Users can browse courses, view details, and manage courses after logging in.

---

##  Key Features

*  Firebase Authentication (Email & Password, Google Login)
*  Landing Page with multiple sections
*  Course Listing Page with search & filtering
*  Dynamic Course Details Page
*  Manage Courses (Protected)
*  Fully Responsive Design
*  Clean UI with Tailwind CSS

---

## Technologies Used

* Next.js (App Router)
* Firebase Authentication
* Tailwind CSS
* React Lucid and React Icons

---

##  Route Summary

| Route         | Description                |
| ------------- | -------------------------- |
| /             | Home Page                  |
| /about        | About Page                 |
| /courses        | All Courses                |
| /courses/[id]   | Course Details             |
| /courses/manage | Manage Courses (Protected) |
| /login        | Login Page                 |
| /register     | Register Page              |

## ⚙️ Setup Instructions

1. Type the terminal:

```
npx create-next-app@latest
```

2. Follow Instruction:

```
Follow the Instruction and click Yes OR No
```

3. Run the project:

```
npm run dev
```

4. Setup Firebase:

* Create a Firebase project
* Add your config in `firebase.js`
